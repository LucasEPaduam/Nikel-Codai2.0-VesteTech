# Nikel

Nikel é um sistema web para controle financeiro pessoal, desenvolvido para facilitar o gerenciamento de entradas, saídas e saldo de contas de forma simples, visual e responsiva.

## Funcionalidades

- Cadastro de lançamentos financeiros (entradas e saídas)
- Visualização do saldo total
- Listagem das últimas entradas e saídas
- Modal para adicionar novos lançamentos
- Navegação entre páginas de Home e Transações
- Sistema de login/logout e permanecer logado preservando a sessão
- Interface responsiva e moderna com Bootstrap 5
- Ícones intuitivos com Bootstrap Icons

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript puro
- Bootstrap 5

## Detalhes Técnicos

- Os dados são armazenados no `localStorage` do navegador, garantindo persistência entre sessões.
- O layout é totalmente responsivo, adaptando-se a dispositivos móveis e desktops.
- O código JavaScript está modularizado por página para facilitar manutenção.
- O sistema utiliza modais do Bootstrap para UX aprimorada.
